**核心类型：**

```
feat: ✨ 引入新功能
```

```
fix: 🐛 修复一个 Bug
```

```
docs: 📝 仅修改文档
```

```
style: 🎨 修改代码格式
```

```
refactor: ♻️ 代码重构
```

```
perf: ⚡️ 提升性能
```

```
test: ✅ 添加或修改测试
```

```
build: 🛠 影响构建系统或外部依赖
```

```
ci: ⚙️ 更改 CI 配置和脚本
```

```
chore: 🔧 其他杂项更改
```

```
revert: ⏪ 撤销之前的提交
```

**其他常用类型：**

```
init: 🎉 初始提交
```

```
wip: 🚧 正在进行中
```

```
security: 🔒 修复安全问题
```

```
remove: 🔥 移除代码或文件
```

```
config: 🔧 修改配置文件
```

```
ui: 💄 更新 UI 和样式
```

```
release: 🔖 发布新版本
```
