启动宿主浏览器打开 url
前提是挂载了 /proc

[[越狱]]

```bash
sudo chroot /proc/1/root /system/bin/sh -c "export PATH=/system/bin; /system/bin/am start --user 0 -a android.intent.action.VIEW -d 'https://bing.com' > /dev/null"
```

### 包装脚本

创建文件 `/data/local/tmp/arch/usr/local/bin/open-in-android`，755，root:root

```bash
#!/bin/sh

URL="$1"

if zenity --question --title="安全确认" --text="即将尝试在 Android 中打开以下链接：\n\n<b>$URL</b>\n\n是否继续？"; then
    chroot /proc/1/root /system/bin/sh -c "export PATH=/system/bin; /system/bin/am start --user 0 -a android.intent.action.VIEW -d \"$URL\" > /dev/null"
else
    exit 1
fi
```

创建文件 `/data/local/tmp/arch/etc/sudoers.d/open-in-android`，640，root:root

```ini
# 允许 yumeka 免密运行 Android 浏览器包装脚本
yumeka ALL=(root) NOPASSWD: /usr/local/bin/open-in-android *
```

创建 .desktop

```bash
mkdir ~/.local/share/applications/ && nvim ~/.local/share/applications/android-browser.desktop
```

写入

```ini
[Desktop Entry]
Name=Android Browser
Comment=通过 'am start' 在 Android 中打开
Exec=sudo /usr/local/bin/open-in-android %u
Terminal=false
Type=Application
MimeType=x-scheme-handler/http;x-scheme-handler/https;
Categories=Network;WebBrowser;
```

更新 .desktop 数据库

```bash
update-desktop-database ~/.local/share/applications
```

设置默认浏览器

```bash
xdg-settings set default-web-browser android-browser.desktop
```
